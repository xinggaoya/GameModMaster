# GameModMaster v2.0.0 发布说明

🎉 **重大版本更新 - 全新体验**

## 🚀 主要新功能

### 1. 🎯 系统托盘支持
- 新增系统托盘图标，支持后台运行
- 右键菜单：显示/隐藏主窗口、退出应用
- 左键单击托盘图标快速显示主窗口
- 优化应用生命周期管理

### 2. 🎨 全新UI设计重构
- 现代化界面设计，采用卡片式布局
- 新增渐变色彩和圆角设计元素
- 优化视觉效果和交互动画
- 改进响应式布局，适配不同屏幕尺寸
- 统一的视觉语言和设计系统

### 3. 📁 下载目录自定义
- 新增下载路径设置功能
- 支持浏览选择下载目录
- 一键打开下载文件夹
- 自动保存下载路径设置
- 下载完成后可选择自动打开文件夹

### 4. 💾 全新存储系统
- 从localStorage迁移到SQLite数据库
- 更稳定可靠的数据存储
- 支持数据缓存和过期管理
- 自动迁移旧版本数据
- 提升数据查询性能

### 5. 🔄 自动更新功能
- 应用启动时自动检查更新
- 手动检查更新按钮
- 下载进度实时显示
- 支持查看发行说明
- 下载完成后自动安装

### 6. 🌍 国际化支持
- 集成vue-i18n国际化框架
- 支持多语言切换
- 中英文完整翻译
- 自动保存语言偏好设置

### 7. 🌓 主题切换系统
- 支持浅色/深色/系统主题
- 自动跟随系统主题
- 主题偏好设置持久化
- 平滑的主题切换动画

### 8. 📊 改进的下载管理
- 实时显示下载进度
- 支持取消下载任务
- 下载历史记录管理
- 优化下载性能和稳定性

## 🔧 技术改进

### 后端优化
- 重构API结构，提高代码可维护性
- 优化错误处理和日志记录
- 改进训练器管理和下载逻辑
- 新增设置管理模块

### 前端优化
- 重构组件结构，提高复用性
- 优化状态管理（Pinia）
- 改进路由和导航体验
- 增强TypeScript类型安全

### 性能提升
- 减少内存占用
- 优化启动速度
- 改进列表渲染性能
- 缓存策略优化

## 📝 版本信息

- **版本号**: 2.0.0
- **发布日期**: 2025年12月13日
- **支持平台**: Windows
- **最低系统要求**: Windows 10 或更高版本

## 📦 安装说明

1. 下载最新版本的安装包（.msi或.exe）
2. 运行安装程序
3. 按照提示完成安装
4. 启动GameModMaster

## 🔥 快速开始

1. 首次启动时，建议先到"设置"页面配置下载目录
2. 浏览或搜索你想要的游戏修改器
3. 点击下载并安装修改器
4. 从"已安装"列表中启动修改器
